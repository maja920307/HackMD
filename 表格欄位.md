# 表格欄位
---
| Name  | Quantity |
| ----- | -------- |
| Apple | 3        |
| Egg   | 12       |

---
| 指令         | 描述                 | 快捷鍵     |
| ------------ | -------------------- | ---------- |
| 下一個單元格 | 移動到下一個單元格   | Tab        |
| 前一個單元格 | 移動到前一個單元格   | Shift-Tab  |
| 下一列       | 移動到下一列         | Enter      |
| 跳出         | 從表格編輯模式中跳出 | Ctrl-enter |
